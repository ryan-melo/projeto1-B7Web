A overflowpropriedade tem os seguintes valores: visible- Predefinição. O estouro não é cortado. O conteúdo é renderizado fora da caixa do elemento hidden- O estouro é cortado e o restante do conteúdo ficará invisível scroll- O estouro é cortado e uma barra de rolagem é adicionada para ver o restante do conteúdo auto- Semelhante a scroll, mas adiciona barras de rolagem apenas quando necessário Nota: A overflowpropriedade só funciona para elementos de bloco com uma altura especificada. overflow-x overflow-y ---------------------------------------------------------------------- A propriedade flutuante A floatpropriedade é usada para posicionar e formatar o conteúdo, por exemplo, deixar uma imagem flutuar à esquerda do texto em um contêiner. A propriedade float pode ter um dos seguintes valores: left- O elemento flutua à esquerda de seu contêiner right- O elemento flutua à direita de seu contêiner none- O elemento não flutua (será exibido exatamente onde ocorre no texto). Isso é padrão inherit- O elemento herda o valor float de seu pai Em seu uso mais simples, a floatpropriedade pode ser usada para quebrar o texto em torno das imagens. A propriedade clara Quando usamos a floatpropriedade, e queremos o próximo elemento abaixo (não à direita ou à esquerda), teremos que usar a clear propriedade. A clearpropriedade especifica o que deve acontecer com o elemento que está próximo a um elemento flutuante. ---------------------------------------------------------------------- A propriedade clear pode ter um dos seguintes valores: none- O elemento não é empurrado abaixo dos elementos flutuantes à esquerda ou à direita. Isso é padrão left- O elemento é empurrado abaixo dos elementos flutuantes à esquerda right- O elemento é empurrado abaixo dos elementos flutuantes à direita both- O elemento é empurrado abaixo dos elementos flutuantes esquerdo e direito inherit- O elemento herda o valor clear de seu pai Ao limpar floats, você deve combinar o clear com o float: Se um elemento flutuar para a esquerda, então você deve limpar para a esquerda. Seu elemento flutuante continuará flutuando, mas o elemento desmarcado aparecerá abaixo dele na página da web. ---------------------------------------------------------------------- GRID: ---------------------------------------------------------------------- z-index: Lugar do eixo Z o elemento vai aparecer quanto maior o valor, mais para a frente o elemento vai ir ---------------------------------------------------------------------- max e min: tag { color: min(50%, 300px); (escolhe o menor valor para aplicar) fonte: max(50%, 300px); (escolhe o maior valor para aplicar) } ---------------------------------------------------------------------- Função calc: tag { width: calc (50% - 20px); } ---------------------------------------------------------------------- Degradê: background: linear-gradient(red, blue) background: linear-gradient(75deg, red, blue) repeating-linear-gradient()function is used to repeat linear gradients: Radial Gradients: background-image: radial-gradient(shape size at position, start-color, ..., last-color); ---------------------------------------------------------------------- Animações: @keyframes NAME { 0% {} 50% {} 100% {} } animation-name: NAME; animation-duration: 5s; animation-delay: 2s; animation-iteration-count: 2 ou infinite; animation-direction: normal- A animação é reproduzida normalmente (para a frente).
